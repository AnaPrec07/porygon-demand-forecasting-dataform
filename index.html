Project Overview

This repository contains the ETL processes required to power a production-grade machine learning solution for demand forecasting, leveraging Google Dataform, BigQuery, and GCP Model Registry. It enables robust feature engineering, target generation, and reporting for retail time series forecasting, with a focus on Walmart data.

  • Automated data pipelines for feature engineering and target creation
  • Data quality checks and modular SQLX definitions
  • Integration with BI dashboards for actionable insights
  • Reproducible, version-controlled transformations and ML-ready tables
